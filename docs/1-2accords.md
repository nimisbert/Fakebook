## Les différents accords <a name="differents"></a>
Tel que $N$ représente une note en notation anglosaxonne.
| Accords        |     |           |     |           |
|:---------------|:---:|:---------:|:---:|:---------:|
| $N$            | $I$ | $III$     | $V$ |
| $Nm$           | $I$ | $III^{b}$ | $V$ |      
| $N7^{ième}$    | $I$ | $III$     | $V$ | $VII$     |
| $Nm7^{ième}$   | $I$ | $III^{b}$ | $V$ | $VII$     |
| $N7^{ième}m$   | $I$ | $III$     | $V$ | $VII^{b}$ |
